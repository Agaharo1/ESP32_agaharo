# Game using the SSD1306 OLED display and a joystick


The used component (OLED display) was developed by nopnop2002

## How to replicate the project

You need esp32, 0.96 inch OLED display and a Joystick

To display images (BIT MAP), follow the steps on this page: [here](https://mischianti.org/ssd1306-oled-display-draw-images-splash-and-animations-2/)
